About the Developer

Hello! My name is Geetha, and I am a student currently enrolled in the ISM 6225 App Dev for Analytics course at the University of South Florida. I’m passionate about web development, data visualization, and using technology to solve practical problems.

Development Experience

This website was developed as part of Assignment 1 to demonstrate basic web interface design skills. I created a static site focused on automotive repair using HTML for structure, CSS for styling, and JavaScript to simulate user interactions.

I also used Chart.js to create a bar chart that displays average repair costs, and Cyberduck to upload my project to the USF web server (myweb.usf.edu). Everything was built and tested using Visual Studio Code.

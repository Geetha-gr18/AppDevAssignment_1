About the Developer

Hello! My name is Geetha, and I am a graduate student at the University of South Florida, currently enrolled in ISM 6225: Application Development for Analytics. This course has allowed me to explore web technologies and their application in data-driven decision-making.

Website Development Experience

This website was developed as part of Assignment 1 and is centered around the theme of Automotive Repair. It includes a homepage, an About page, a CRUD simulation for managing repair records, and an analytics page displaying a repair cost chart.

To build the site, I used HTML to structure content, CSS to style and unify the look and feel across pages, and JavaScript to simulate interactivity. I also used Chart.js for creating a visual representation of average repair costs and Cyberduck to upload my files to the USF MyWeb server. All coding was done in Visual Studio Code.
